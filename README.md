# RobotsUnions
Dataset for the paper Labor Unions and Robots: Do Employers with More Powerful Organized Labor Automate More? by Larry Liu
Rmd file contains the code. Htm and png are the output files. The three data files end with csv. [Note: data cleaning originated from different data files that were too large to be hosted on Github. To reproduce the analysis, load the three data files then scroll down to the code beginning "## Implement regression"]
